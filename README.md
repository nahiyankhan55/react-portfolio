# Portfolio Website - Kushbula Ahmed Nahiyan Khan

A modern, responsive, and interactive portfolio built with **React**, **Tailwind CSS**, **Framer Motion**, **Material-UI**, and **EmailJS**. It showcases projects, skills, and contact information with animated sections and a professional UI.

---

## **Live Link**

[https://kushbula-ahmed-nahiyan.netlify.app/](https://kushbula-ahmed-nahiyan.netlify.app/)

---

## **Features**

- **Animated Sections**: Smooth scroll animations using Framer Motion for About, Skills, Projects, and Contact.
- **Skills Section**: Card-based skills with icons and colored borders.
- **Projects Section**: Project cards with image, description, tech stack, and links to live site & GitHub.
- **Contact Form**: Integrated with **EmailJS** to send messages directly.
- **Responsive Design**: Works on mobile, tablet, and desktop screens.
- **Dark/Light Mode** ready (via Tailwind classes).

---

## **Technologies**

- **Frontend**: React.js, Tailwind CSS, Framer Motion, Material-UI, React Icons
- **Backend for Contact Form**: EmailJS (no backend server required)
- **Others**: Toast notifications using `react-hot-toast`
